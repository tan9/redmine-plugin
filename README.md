Jenkins Redmine Plugin
=============================

これは何?
---------
[こちら](http://wiki.jenkins-ci.org/display/JENKINS/Redmine+Plugin)で公開されているJenkinsとRedmineを連携するプラグインの改良版です。


変更点
-------

* 変更履歴からチケットへのリンクが、Redmineのバージョンにより異なることへの対応
* ビルド開始時に、Redmineのリポジトリ情報をアップデート
* チケットに、Jenkinsのビルドへのリンクおよびビルド結果を登録
* Wiki記法の削除
* 画面左に表示されるRedmineのアイコンを最新化
